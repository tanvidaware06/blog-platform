# Blog Platform with Comments

## Features

* User Registration
* User Login
* Create Blog Posts
* Edit Blog Posts
* Delete Blog Posts
* Add Comments
* RESTful API using Express.js

## Technologies Used

* Node.js
* Express.js
* HTML
* CSS
* JavaScript

## Run Project

Install dependencies:

npm install

Start server:

node server.js

Server runs on:

http://localhost:5000
